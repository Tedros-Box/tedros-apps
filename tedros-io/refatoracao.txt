### Key Content from Tedros Overview (Incorporate and Organize This into Website Sections, All in Portuguese):

#### Introdução
- Tedros Box é uma solução desktop cliente-servidor em Java 17, com frontend em JavaFX e backend em OpenEJB/Apache TomEE.
- Framework Tedros reduz custos de desenvolvimento e melhora a experiência do usuário ao abstrair processos complexos (ex.: design de layout, permissões, integrações, mapeamento de entidades com ORM).
- Ideal para soluções personalizadas em setores privados ou governamentais, similar a TOTVS ou SAP, com integração nativa a modelos de IA, permitindo que assistentes de IA acessem dados e funções via chatbot para operações mais inteligentes e eficientes.
- Permite foco no core business; UI customizável, gerenciamento de usuários e permissões autônomo.
- Desenvolvido por Davis Gordon Dun (arquiteto de software Java com mais de 12 anos de experiência).

#### Arquitetura do Sistema
- Sistema desktop cliente-servidor que detecta automaticamente apps instalados e gerencia permissões de usuário (apps, módulos, telas, ações).
- Contextos isolados de apps para tolerância a falhas.
- Camadas backend: Persistência (CRUD, paginação, consultas dinâmicas), Negócios (processos centrais), Serviços (transacionais, chamadas inter-apps), Controle (chamadas frontend com validação de permissões).
- Comunicação seamless entre apps no frontend e backend.

#### Componentes do Framework
- Importação de dados via Excel/CSV com guias para usuários.
- Filtros, relatórios (com logotipos da empresa).
- Agendamento/envio de e-mails.
- Tarefas CRON.
- Gerenciamento centralizado de arquivos/imagens.
- Construção dinâmica de formulários/telas/tabelas/filtros com interceptação em runtime (pré/pós-construção em níveis de app/módulo/tela/campo).
- Internacionalização (inglês/português nativos; fácil adicionar mais).
- Customização visual (cores, opacidade, fundos, temas).
- Criação de gráficos.
- Desenvolvimento de módulos.
- Gerenciamento de processos em background.
- Integração entre apps (ex.: rastrear origem de registros, exibir tela de origem).
- Modais e mensagens para usuários.
- Validação de permissões.
- Integração com chatbot de IA para acessar recursos do sistema (ênfase: o chatbot usa IA para tarefas operacionais, ajuda no sistema, análises de dados como e-mails de marketing baseados em cruzamento de clientes/estoque, verificações de consistência em demandas, revisões de código, e mais; destaque a integração nativa com modelos de IA para eficiência inteligente).
- Exibição/manipulação de mapas geográficos.
- Navegação entre apps/módulos/telas.
- E mais para flexibilidade.
